# Forever World .mcpack Repo



<p>Just a simple repo for friends to grab a modpack easier.</p>

<p>Modpacks will be added once they are stable enough to run. I will also add more versions if I add or delete or modify and packs</p>





<h2>How to Install / Options</h2>



<p>Installation Steps:</p>

<ol>

<li>Click on the folder of the version you are trying to install</li>

<li>Click on the folder of the latest version of the modpack</li>

<li>Choose if you want a Base or Clean pack and click it</li>

<li>In the top right-ish, click "Raw" (This should download the file)</li>

<li>Go to wherever the .mrpack file is stored and run it</li>

</ol>



<p>If you have Modrinth downloaded, it should automatically open it and download the instance to your library.</p>



<p>Base vs Clean</p>

<ul>

<li>Base - Has my settings, worlds and mod configurations</li>

<li>Clean - Has No worlds and very little mod configurations</li>

</ul>



<h2>Mod Configurations</h2>

<p>

I highly recommend you explore the mods using the in game mod menu. Frostbyte's Inventory gives a lot of options for more than just your HUG/GUI. Subtle Effects adds a lot of different particle effects to the game, all of which can be customized. Particle Rain and Sounds are other mods you can configure to your liking. Take some time to tinker and figure out what settings you like!

</p>
